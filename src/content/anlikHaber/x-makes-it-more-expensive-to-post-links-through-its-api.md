X, API üzerinden bağlantı paylaşmanın maliyetini belirgin biçimde artırdı.

TechCrunch’ın haberine göre şirket, link paylaşımı için API ücretini 0,01 dolardan 0,20 dolara çıkardı. X, bu değişikliğin spam ve kötüye kullanım yollarını azaltmak için yapıldığını söylüyor.

Yeni tarife, bağlantı gönderen uygulama ve otomasyon araçları için maliyetleri doğrudan yükseltecek. Bu da özellikle API üzerinden içerik dağıtan geliştiriciler açısından yeni bir hesaplama anlamına geliyor.

X’in son hamlesi, platformun erişim ve kullanım şartlarını daha sıkı bir çerçeveye çektiğini gösteriyor. Değişikliğin pratik etkisi, link yoğun çalışan üçüncü taraf araçlarda kısa sürede hissedilecek.
