# Runtime data

Bu klasördeki içerikler çalışma zamanı çıktılarıdır.

Alt klasörler:
- `raw/` → ham feed kayıtları
- `normalized/` → normalize edilmiş kayıtlar
- `queue/` → editoryal kuyruk öğeleri
- `logs/` → çalışma logları
- `published/` → publish ilişki/veri artıkları

Bu veriler repo mantığında kaynak kod değildir.
Commit'e dahil edilmemeli, yalnız klasör yapısı referans olsun diye `.gitkeep` tutulur.
